# Get Computer SID

## Abstracts

* Get Computer SID (a.k.a Machine SID)

## Requirements

### Common

* Powershell
* CMake 3.0.0 or later
* C++ Compiler supports C++11

### Windows

* Visual Studio

## How to build?

````shell
$ pwsh build.ps1 <Debug/Release>
````

## How to use?

````bat
$ install\win\bin\Test.exe
Computer SID: S-1-5-21-1366212534-334562847-3460492818  
````
