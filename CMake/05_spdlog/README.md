# spdlog from CMake

## Abstracts

* Build and link spdlog by cmake

## Requirements

## Common

* Powershell
* CMake 3.20.0 or later

### Windows

* Visual Studio

### Ubuntu

* g++
* build-essential
* gdb

### OSX

* Xcode

## Dependencies

* [spdlog](https://github.com/gabime/spdlog)
  * v1.11.0
  * MIT License

## How to usage?

````shell
$ pwsh build-spdlog.ps1  <Debug/Release>
$ pwsh build.ps1  <Debug/Release>
[2023-01-24 22:12:36.293] [info] Hello, World!
````
