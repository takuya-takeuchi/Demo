# curl from CMake

## Abstracts

* Build and link curl by cmake
  * static build
* Minimal example program

## Requirements

### Common

* Powershell
* CMake 3.0.0 or later
* C++ Compiler supports C++11

### Windows

* Visual Studio 2019

### Ubuntu

* g++
* libssl-dev

### OSX

* Xcode

## Dependencies

* [curl](https://github.com/curl/curl)
  * 7.87.0
  * MIT/X Derivative License
