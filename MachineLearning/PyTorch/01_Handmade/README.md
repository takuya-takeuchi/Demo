# Handmade Neural Network

## Abstacts

* Simple Neural Network for classification
  * Train Cifar10

## Requirements

* Python 3.6 or later
* PyTorch 1.10.2

Please check [requirements.txt](./requirements.txt)

## How to usage?

````cmd
$ python -m pip install -r requirements.txt
````

### Train

````cmd
$ python train.py --epoch 100 --batchsize 128
````
