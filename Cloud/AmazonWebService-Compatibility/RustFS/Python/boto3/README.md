# boto3

## Requirements

### Common

* Python3 or later

## Dependencies

* [boto3](https://github.com/boto/boto3)
  * 1.43.6
  * Apache 2.0 License

## Setup

### Windows

````bat
$ python3 -m venv .venv
$ 
````

### Linux and OSX

````bat
$ python3 -m venv .venv
$ source .venv/bin/activate
$ python -m pip install -r requirements.txt
````
