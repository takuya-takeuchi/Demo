# Open3D

## Requirements

### Common

* Python 3.7 or later

## Dependencies

* [open3d](https://github.com/isl-org/Open3D)
  * v0.18.0
  * MIT license

## How to build?

````bat
$ python3 -m venv .venv
$ source .venv/bin/activate
$ python -m pip install pip --upgrade
$ python -m pip install open3d==0.18.0
````
