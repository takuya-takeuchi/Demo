# LlamaIndex

## Abstracts

* Constrcut RAG (Retrieval-Augmented Generation) by LlamaIndex with various LLM and vector database

## Requirements

### Common

* Python 3.10 or later

## Dependencies

* [LlamaIndex](https://github.com/run-llama/llama_index)

* [LlamaIndex Llms Integration: Ollama](https://github.com/run-llama/llama_index)

* [LlamaIndex Vector_Stores Integration: Opensearch](https://github.com/run-llama/llama_index)

## How to use?

### Windows

````shell
$ python3 -m venv .venv
$ .\.venv\Scripts\activate
$ python -m pip install -r requirements.txt
````

### Linux and OSX

````shell
$ python3 -m venv .venv
$ source .venv/bin/activate
$ python -m pip install -r requirements.txt
````
