# Hello world

## Requirements

### Windows

* [Emscripten SDK](../emsdk)
  * MIT and Expat Licence
* [ninja](https://github.com/ninja-build)
  * Apache-2.0 Licence

## How to use?

### Windows

Install `Emscripten SDK` and add location `ninja` to `Path` envrionmental variable.

## How to build?

````bat
$ eccmake ..
$ eccmake --build . --config Release
````
