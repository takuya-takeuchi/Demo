## Abstracts

* How to transpose tensor which is created from float pointer
  * Not using template parameter

## Requirements

* Powershell 7 or later
* CMake 3.9.0 or later

## Windows

* Visual Studio 2022

## Linux

* GCC

## OSX

* XCode
