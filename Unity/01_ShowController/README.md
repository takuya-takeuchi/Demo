# Show Controller

## Abstacts

* How to show oculus controller
* How to show line objects like lazer

## Hardware Requirements

* Oculus Quest 2

## Software Requirements

* Unity
  * 2021.3.5f1
* Oculus Integration
  * 41.0

## How to build

You must import Oculus Integration from Package Manager before build.
