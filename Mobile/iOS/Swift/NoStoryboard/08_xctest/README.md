# Simple xctest project

## Abstracts

* A simple framework, xctest and host app projects
  * Framwork and xctest project have bundle identifer but these identifers are not registered into Apple Developper Program. You need to take care of only host app project.

## Requirements

* Xcode 14.0 or later
* Apple Developer Account with Apple Develop Program
