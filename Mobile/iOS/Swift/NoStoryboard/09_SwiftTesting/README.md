# Simple Swift Testing project

## Abstracts

* A simple framework, swift testing and host app projects
  * Framwork and swift testing project have bundle identifer but these identifers are not registered into Apple Developper Program. You need to take care of only host app project.

## Requirements

* Xcode 15.0 or later
* Apple Developer Account with Apple Develop Program
