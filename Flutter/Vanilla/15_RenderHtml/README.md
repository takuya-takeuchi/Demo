# Render HTML as widget

## Abstracts

* Reder html content into widget
  * Use `flutter_widget_from_html` package
    * Hook `h1`, `h2`, `h3`, `li`, `ul` and `ol` tags. Especially, customize color of symbol of list.

## Dependencies
