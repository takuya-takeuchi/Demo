# Vlc Video Player

## Abstracts

* How to play network, local movie and ip camera stream by using `flutter_vlc_player` package

## Dependencies

* [flutter_vlc_player ](https://github.com/solid-software/flutter_vlc_player/)
  * BSD-3-Clause license
