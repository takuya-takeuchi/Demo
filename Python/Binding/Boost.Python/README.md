# Boost.Python

## Abstracts

* Binding C++/Python by Boost.Python

## Requirements

### Common

* Powershell
* CMake 3.12.0 or later
* C++ Compiler supports C++17

### Windows

* Visual Studio

### Ubuntu

* g++

### OSX

* Xcode

## Dependencies

* [Boost](https://www.boost.org/)
  * v1.86.0
  * Boost License

## How to usage?

````shell
$ pwsh download.ps1 <BoostVersion, like 1.86.0>
$ pwsh build.ps1 <BoostVersion, like 1.86.0>
````
